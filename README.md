# 💱 Currency Converter

A responsive Currency Converter built using **HTML, CSS, and JavaScript**. It fetches live exchange rates from a public API and updates country flags dynamically.

## ✨ Features

- 🌍 Convert between 150+ currencies
- 📈 Live exchange rates
- 🚩 Dynamic country flags
- 📱 Responsive design
- ⚡ Fast API requests
- 🖥 Clean UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Currency API
